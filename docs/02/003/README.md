## **背景概述**

现有的音乐智能玩具大多通过对音乐的细分和场景化推荐，满足不同圈层用户在各种典型场景下的差异化需求。结合不同的交互形式为人们提供 了更方便欣赏、学习、使用的一条新途径，满足了各类人群、在各种环境场合下对音乐的个性化需求。



### 当下音乐展现形式
  当下的音乐展现形式多基于不同音乐平台，以不同场景进行音乐内容的细分及推荐，从音乐的类型及风格出发提供不同的演奏及交互形式以满足不同圈层用户在不同场景下的差异化需求。
